## 🏗️ Stack

| Component | Choice |
|-----------|--------|
| Document class | `moderncv` (banking style) |
| Color theme | blue |
| Icons | `fontawesome5` |
| Compiler | **XeLaTeX** (required) |

> ⚠️ **Important:** `fontawesome5` + `moderncv` banking require **XeLaTeX**.
> Using `pdflatex` produces degraded rendering (missing colors, fallback fonts).

## 🚀 Build locally

### Prerequisites

```bash
# Ubuntu / Debian — full texlive (recommended)
sudo apt-get install texlive-full

# Minimal (adjust if packages are missing)
sudo apt-get install texlive-xetex texlive-latex-extra texlive-fonts-extra
```

### Compile

```bash
xelatex Resume.tex
```

Second pass (if cross-references change):

```bash
xelatex Resume.tex && xelatex Resume.tex
```

### Output

`Resume.pdf` in the same directory.

## 🤖 CI/CD

Both pipelines trigger on every push to `master`:

| Platform | Config | Artifact |
|----------|--------|----------|
| **GitHub Actions** | `.github/workflows/build-pdf.yml` | PDF artifact (90 days) + PNG preview committed |
| **GitLab CI** | `.gitlab-ci.yml` | PDF + PNG artifacts (1 year) |

The compiled PDF is available as a CI artifact after each build.
On **tagged releases** (`git tag vX.Y && git push --tags`), a GitHub Release is created with the PDF attached.
